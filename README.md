# 🌦️ Weather App

A simple and modern Android weather application built with Kotlin that provides real-time weather information.

## ✨ Features

- Search weather by city
- Real-time weather updates
- Temperature and weather conditions
- Clean and responsive UI
- Fast and lightweight application

---

## 🛠️ Technologies Used

- Kotlin
- Android SDK
- REST API
- Retrofit
- MVVM Architecture

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/ithassan/weather_app.git
```

### Open in Android Studio

1. Open Android Studio
2. Select Open Project
3. Sync Gradle
4. Run the app
